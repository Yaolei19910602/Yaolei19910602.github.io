##### 基本介绍
- 源代码放在 `blog` 分支，编译后代码部署在 `master`分支
- 克隆当前仓库进入 `blog` 分支进行写作

##### 基本命令 
- 本地启动命令 `yarn start` 开始进行写作
- 创建新文章 `yarn new [文章名称]`
- 将代码push到远程仓库 blog 分支
- 部署命令本地执行 `yarn deploy`
